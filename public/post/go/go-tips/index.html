nums := []int{0,1,2,3}

// 1. 这个挺容易理解的
nums = append([]int{1,2,3}, nums[:0])
fmt.Println(nums)   // [1 2 3]

// 2. 奇怪的来了
a := nums[4]    // panic: index out of range
nums = append([]int{1,2,3}, nums[4:])  // 编译通过
fmt.Println(nums)   // [1 2 3]
nums = append([]int{1,2,3}, nums[5:])  // panic: index out of range
fmt.Println(nums)   // [1 2 3]

在使用 ... 打散切片时，切片索引可以是切片长度！！但不能更大！
